В чем разница между выражениями WHERE и HAVING

Параметры
WHERE
HAVING
Реализовано в Строковых операциях Столбцовах операциях
Относится к Отдельной строке Суммированной строке или группам
Используется для Выборке конкретных данных из определенных строк, согласно заданным условиям Выборке всех данных и отделения тех из них, которые соответсвуют заданным условиям
Агрегатные функции Не может содержать Может содержать
Операторы Может использоваться с SELECT, DELETE и UPDATE Не может содержаться без оператора SELECT
Выражение GROUP BY Идет после выражения WHERE Идет перед выражением HAVING
